# AgenticCoder: Multi-Agent AI Coding System 🚀

An end-to-end coding agent portfolio project demonstrating state-of-the-art (SOTA) agentic AI concepts.

## 🌟 Market Trending Concepts Showcased
- **Multi-Agent Orchestration (LangGraph)**: State-machine based workflow with Planner, Coder, and Tester agents.
- **Hybrid Retrieval (RAG + GraphRAG)**: Combines vector similarity search (ChromaDB) with codebase structure graphs (NetworkX).
- **Model Context Protocol (MCP)**: Standardized, secure tool execution.
- **Observability**: Built-in LangSmith tracing.

## 🛠️ Tech Stack
- Python 3.11+
- LangGraph / LangChain
- Gemini Models
- ChromaDB & NetworkX

## 🚀 Getting Started

1. Clone the repository
2. Set up a virtual environment: `python3 -m venv venv && source venv/bin/activate`
3. Install dependencies: `pip install -r requirements.txt`
4. Copy `.env.example` to `.env` and add your API keys.
5. Run the orchestrator: `python main.py`
